# Coinbase
Diseño web clon de la página oficial de coinbase, la cual fue realizada bajo la tutoría de OpenBootCamp como metodología para interiorizar los conceptos aprendidos en el curso de HTML y CSS

## Nota:
Este proyecto utiliza Sass, para ello, debes modificar los archivos scss del proyecto y después compilarlo a css de la siguiente manera:

`sass --watch sass/index.scss css/styles.css`

Si no tienes instalado sass, puedes hacerlo de la siguiente manera.

`npm install -g sass`

para más información acerca de SASS visita el siguiente [enlace](https://sass-lang.com/)
